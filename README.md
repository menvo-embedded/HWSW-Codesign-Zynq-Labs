# HW/SW Codesign Labs on Zynq SoC

## Giới thiệu

Repository này dùng để lưu trữ các bài thực hành môn HW/SW Codesign trên nền tảng Zynq SoC.  
Các bài lab được thực hiện bằng Vivado 2019.2 và Vitis 2019.2, tập trung vào thiết kế hệ thống phần cứng/phần mềm, kết nối AXI GPIO và lập trình C điều khiển ngoại vi.

## Thông tin

- Môn học: HW/SW Codesign
- Nhóm: 18
- Công cụ: Vivado 2019.2, Vitis 2019.2
- Board: ZedBoard Zynq Evaluation and Development Kit
- Mục tiêu: Lưu trữ source code, file thiết kế, file export và báo cáo cho các bài lab.
